## Fullstack Todo App with TanStack

### Technologies

#### Frontend

| Technology            | Purpose                              |
| --------------------- | ------------------------------------ |
| React 19              | UI library                           |
| TypeScript            | Type safety                          |
| Vite                  | Build tool & dev server              |
| TanStack Router       | File-based routing                   |
| TanStack Query        | Server state & data fetching         |
| Axios                 | HTTP client                          |
| Tailwind CSS v4       | Styling                              |
| shadcn/ui             | Component library                    |
| clsx + tailwind-merge | Class merging                        |
| Lucide React          | Icons                                |
| openapi-typescript    | OpenAPI → TypeScript type generation |

#### Backend

| Technology                   | Purpose                 |
| ---------------------------- | ----------------------- |
| .NET 10                      | Runtime & framework     |
| ASP.NET Core Minimal APIs    | HTTP endpoints          |
| Entity Framework Core 10     | ORM                     |
| PostgreSQL (Npgsql)          | Database                |
| Microsoft.AspNetCore.OpenApi | OpenAPI spec generation |
| Scalar                       | API docs UI and testing |
